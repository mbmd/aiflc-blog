## 🛠️ For Maintainers

<details>
<summary>Local development & publishing</summary>

### Prerequisites
- [Hugo Extended](https://gohugo.io/installation/) (v0.158+)
- [Git](https://git-scm.com/)

### Run locally
```bash
hugo server -D
```
Preview at http://localhost:1313 (includes drafts).

### Add a post
```bash
hugo new content content/methodology/my-new-post.md
```
Edit the file, set `draft: false` when ready, then push.

### Publish
Push to the `main` branch. GitHub Actions builds and deploys automatically to GitHub Pages.

### Content structure
```
content/
  engineering/    How AIFLC works (deep dives)
  product/        Package launches, features, roadmap
  methodology/    Thought leadership, process philosophy
  open-source/    Community, licensing, contributions
  tutorials/      Step-by-step guides
  changelog/      Per-release notes
```

</details>
